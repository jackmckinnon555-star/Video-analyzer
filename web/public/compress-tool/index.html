TRA Video Analyzer · local compression tool

Compress large videos locally

The in-browser compressor tops out near 2 GB. Bigger videos need a quick native pre-pass — runs in 30-90 s for a 3-hour talk and produces a file the uploader accepts directly.

1. Install ffmpeg (once)

# Windows
winget install Gyan.FFmpeg

# macOS
brew install ffmpeg

# Linux (Debian / Ubuntu)
sudo apt install ffmpeg

Verify: open a terminal and run ffmpeg -version.

2. Download the script

compress.ps1 Windows · PowerShell compress.sh macOS / Linux / Git Bash README.txt Full instructions

3. Run it

Windows — PowerShell

# Replace with the actual path to your video
.\compress.ps1 "C:\path\to\video.mp4"

# Or drag-drop the video onto compress.ps1 in Explorer.
# If PowerShell blocks unsigned scripts:
powershell -ExecutionPolicy Bypass -File .\compress.ps1 "video.mp4"

macOS / Linux / Git Bash

chmod +x compress.sh
./compress.sh path/to/video.mp4
Output lands alongside the original as name-compressed.mp4 (or .m4a for multi-hour videos). Drop that file into the web uploader — it's already under 50 MB so the browser compressor will skip.

Prefer a one-liner?

If you're comfortable with ffmpeg, these settings match what the web tool produces (audio-only; works for any length): 

ffmpeg -i input.mp4 -vn -c:a aac -b:a 32k -ac 1 -movflags +faststart output.m4a

For short clips where you want playable video:

ffmpeg -i input.mp4 \
  -c:v libx264 -preset ultrafast -b:v 300k -maxrate 300k -bufsize 600k \
  -vf "scale='min(640,iw)':'-2'" \
  -c:a aac -b:a 32k -ac 1 \
  -movflags +faststart output.mp4
